What is ACASE?

ACASE (Automated Crawler and Scrapper Engine) is a Automated Scrapper Bot capable of search information from specialized websites recommended by the company exposing this data to an API. In addition ACASE has a user interface where all the obtained data can be displayed to the user.

Solving Inspyra's problem

Vigitech is a service from Inspyra's Company that delivers reports about digital transformation to the users.
Currently the human scrapper expends 8 hours to get 100 resources, ACASE add automatization to this recipe, giving to Vigitech the same information in 0,1% of the current time.

How ACASE works

ACASE visit the websites given by Inspyra's team, collect all the data and deliver it on the ACASE API, store the data in a database and finally supplies the results on ACASE Intranet.
